CSSDOC

Jan 16 2014 css

CSSDOC采用的是DocBlock，一个借鉴自PHPDocumentor项目的术语。一段DocBlock是一段可供人类和机器读取的数据块；

一段DocBlock以 /** 开头，以空格加 */ 结尾(像正常CSS注释一样)。而它的每一行必须以空格加星号(*)开头，否则将被文档生成器忽略。标签以@开始，紧跟着名字，中间不带空格。标签的描述或值应该放在标签名称之后，前面用空格分开。

CSSDOC建议，应该在每个CSS文件的开头包含一个文件注释(file comment)，内容是与整个文档相关的元数据。在文件注释中，应该包括文件的标题、描述、标签等信息；在文件注释后面，可加入零或多个区块注释(section comment)，对样式表进行内容相关的区块划分；

下面是一些常用的CSSDOC标签：

/**
 * @project
 * @version
 * @package
 * @author
 * @copyright
 * @cssdoc
 * @license
 * @colordef
 * @subpackage
 * @section
 * @subsection
 * @bugfix
 * @workaround
 * @affected
 * @css-for
 * @see
 * @link
 * @valid
 * @todo
 * @date
 * @lastmodified
 * @since
 * @revision
 * @since-revision
 */

下面再给一个实例：

/**
 *@project jayself
 *@author jay.zhang
 *@copyright blog.jayself.com
 *@colordef #333; main text
 *@colordef #2c3e50; links
 */
