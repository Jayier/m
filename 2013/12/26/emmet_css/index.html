用 Emmet 书写 css

Dec 26 2013 emmet

对于css的emmet语法，多数情况下直接取第一个字母，如果是多个单词连接，一般取每个单词的首字母或取第一个单词的首字母和第二个单词的前两字母；

对于sublime，如果装了emmet插件，一般输入一个字母后就会有提示：

属性语法：

m → margin: ;

m10 → margin: 10px;

m10-20 → margin: 10px 20px;

m10--20 → margin: 10px -20px;

m10-20-30 → margin: 10px 20px 30px;

m10-20-30-40 → margin: 10px 20px 30px 40px;

m10em-20 → margin: 10em -20px;

m10em20 → margin: 10em 20px;

bd → border:

bd2-s-#cc → border: 2px solid #ccc;

bd2-da-#ccc → border: 2px dashed  #ccc;

bd2-do#ccc → border: 2px dotted #ccc;

单位：

p → %

e → em

x → ex

w10 → width: 10px;

w10p → width: 10%;

w10e → width: 10em;

w10x → width: 10ex;

颜色：

#1 → #111111

#e0 → #e0e0e0

#fc0 → #ffcc00

c#1 → color: #111111;

c#e0 → color: #e0e0e0;

c#fc0 → color: #ffcc00;

bd1-s-#c → border: 1px solid  #ccc;

不带单位的写法：

lh2 → line-height: 2;

fw400 → font-weight: 400;

z1 → z-index: 1;

op.5 → opacity: .5;

!important：

p10!+m10! → padding: 10px !important;
margin: 10px !important;

最后来一个综合实例：

posa+r0+b28+h100p+w100+bgc#fe:

position: absolute;
right: 0;
bottom: 28px;
height: 100%;
width: 100px;
background-color: #fefefe;
